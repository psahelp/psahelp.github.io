11.28 Release History
When upgrading to release 320 or above Con8 should also be upgraded
Release 329 – 24 January 2018
  • With maximum horizontal scrolling turned on moving to the right stopped after about 10 screen widths – corrected
  • The number of days displayed could vary from number selected depending on width of form.
  • Added code to handle timeout error in SV_Queue routine
  • Changing resource value from menu only set resavstring if the resource was global
  • on selecting SFDC tab of Task data put focus on quantity field - faster updates for manual changes
  • Add option to clear remaining make ready on start of run – simpler manual task updates
  • Minor changes to the convertor to add option to import multiple PSA2 ini files to PSA3 settings table - also ensure that routine to calculate remaining time gets run after upgrade from PSA2
  • An obscure object 91 error was shutting PSA3 down from Schedule Control – this has now been diagnosed and corrected
  • Delete option was disabled for tasks if extended notes was turned on
  • Handle corrupted local db found on first synchronise after open
  • Indicator for linked parent or child to report the linked job number
Release 328 – 8 December 2017
  • • Resources update direct from right click menu on planning board or Dynamic work list could be unreliable especially for limited access users – this should be the definitive version
  • • Add new coating type ADHESIVE and VARNISH when selecting attributes from the WI_Rcoat table.
  • • Fix problem printing from SQL database when using .Net version of Crystal Reports
  • • Add performance logging for maintaining non working periods and queue control
  • • On duplicates screen only show material groups that are in use
  • • Exclude issued quantity from materials screen – required quantity was being overstated
