Permissions tab

The level of access associated with the user’s password can be controlled depending upon which options are selected.

Allow order maintenance – for Limited Access becomes ‘Allow order browse’ – normally checked as turning it off prevent any access to the order maintenance screen.

Allow shift maintenance – allow add, delete and change of non working periods – not available to Limited Access users

Allow customer maintenance – Allows add, delete and change to customer lists - not available to Limited Access users

Allow resource update – Allows access to resource maintenance – see Resource tab below

Allow task update – allows access to task maintenance

Allow drag-drop – allows tasks to be dragged and dropped on the planning board provided permission is available on both source and target work centres

Allow PsaConnect polling – allows PsaConnect to be polled after an update so that changes are sent as soon as possible to SFDC. With this disabled the ‘Update Plan’ screen will close automatically after the update completes

Show Utilities – allows access to all the options on the Utilities menu - not available to Limited Access users

Show Reports – allows access to the reports menu

Show Set-up tables – allows access to the Configuration / Tables menu - not available to Limited Access users

Show Shop floor enquiry – obsolete – there used to be a separate screen for SFDC details – all users can see the planning board and see SFDC status

Allow hold / release – allows hold and release of tasks/orders

‘Password on Update’ - when checked will prompt the user for a password before any updates to the live plan are permitted.

Allow update messaging – allows a What-If user to add a message in the ‘Update Plan’ screen that is visible to other What-If users. This is short term as only the last ten messages are retained

Allow notes update – allows access and update to order level notes attached to the notes tab of order maintenance

Allow What-If mode - Allow Auto-Sync mode – Either What-If or Auto-Sync must be allowed. In general it is recommended that ‘Fully Amendable’ schedulers only use ‘What-If mode and all other users use ‘Auto-Sync mode and no change is allowed. This avoids possible confusion.
