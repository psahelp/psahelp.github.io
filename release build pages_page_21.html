Scheduling
Set up of Schedule Control – preferences – refer to detail screens
The schedule control screen is used to plan, sequence and schedule all orders. New orders, either entered manually or imported from Radius, will have a default planning status of ‘Unplanned’ and will be listed on the unplanned tab. Orders which have been ‘Planned’ and which are available for scheduling will appear on the planned tab.
All Planned orders have a hidden sequence number that controls the order they appear in on the planned tab.
Right clicking on an order opens a context sensitive menu. Details of a selected order are shown by pressing the ‘Detail’ button. This loads the order maintenance screen showing the relevant order. The ‘Search’ button can be used to locate any order that appears in either grid.
Orders can now be unplanned using the ‘Unplan Order’ from the context sensitive menu option’. The selected order together with its components and tasks will have its planning status set back to ‘Unplanned’ and all schedule dates and times will be deleted.
An additional button ‘Select’ has been added to enable a group of unplanned orders to be selected and then copied to the Register. A From and To date may be selected or all orders with an attribute within a specified range. The Attribute may be selected by typing the first letter – you do not need to use the drop down arrow. Duplicate orders are not copied.
Within the unplanned order window double-clicking an order will copy it to the Register. Dragging the mouse pointer to highlight a range of orders followed by the copy button will transfer them to the Register.
Within the planned order window an order may be unplanned by double-clicking. This pops up a message box prompting for unplanning.
