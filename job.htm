job

FIELD NAME                    TYPE      LENGTH    DESCRIPTION

 

job                           Long      4         task number

Description                   Char      30        task description

runlength                     Long      4         estimated length of prodution run in minutes

mrlength                      Long      4         estimated make ready time in minutes

resource                      Char      12        work centre

colour                        Char      10        colour for task ticket - redundant - uses value from ordheader

status                        Char      1         U - unplanned; P - planned; F - firm; C - complete

customer                      Char      25        customer name - redundant - uses value from ordheader

ordernumber                   Char      15        Radius production order - trailing @ indicates duplicate order reimported with changed details - special value 99999999 used for non working periods ans MAINT for work centre maintenance tasks

quantity                      Long      4         planned production quantity including run waste

orig_numup                    Integer   2         original number up - only populated if the number up changes

fixed                         Char      1         Y - task is firm and should not normally be moved

processtype                   Char      6         task process type - should match work centre process type

component                     Integer   2         component number

taskwaittime                  Long      4         wait time in minutes - used for calculating earliest start of following task

taskmovetime                  Long      4         move time in minutes (allowance for drying, transport etc) - used for calculating earliest start of following task

tasklength                    Long      4         sum of mrlength and runlength

displaylength                 Long      4         difference between strtime and edtime - may differ from tasklength because of work centre efficiency and non working periods

strtime                       Long      4         scheduled task start time (held as minutes from 1/1/1995 - see also ptaskstartdt

edtime                        Long      4         scheduled task end time (held as minutes from 1/1/1995 - see also ptaskenddt

origend                       Long      4         stores original mrlength when this has been reduced to zero by the follow on process

record                        Long      4         unique alternative index used for linking prior and following tasks

prosequence                   Integer   2         process sequence copied from work centre

lateststrt                    Long      4         calculated latest start date/time of task to achieve delivery date - see also p_lateststrt

date_completed                Long      4         actual time task is completed-held as minutes from 1/1/1995

job_type                      Char      10        for grouped tasks only holds the order number of the group task - see also multigroup

constrained                   Char      1         Y = the scheduled start time has been affected by a resource constraint

actual_starttime              Long      4         actual time task is started-held as minutes from 1/1/1995

runstatus                     Char      1         X - not started; M - make ready; N - make ready stopped; P - make ready complete; S - run started; H - stopped; C - Complete

last_stoptime                 Long      4         used in manual task update only

mr_starttime                  Long      4         start of most recent make ready event

stop_reason                   Char      20        last recorded stop reason - from SFDC

overlap_style                 Char      1         S - serial - following tasks start after end of this task; P - parallel - following tasks can start some time after the start of this task

start_violation               Char      1         Y = split task - original definition obsolete

end_violation                 Char      1         obsolete

task_sequence                 Integer   2         normally the same as task number (job) - establishes precedence of tasks of same component on same work centre

quantity_completed            Long      4         quantity completed

mr_endtime                    Long      4         actual time make ready completed

speed                         Double    8         speed in units/hr

Upstatus                      Long      4         update reference number

delivery_date                 Date/Time 8         task delivery date (date only) - only relevant if greater than ordheader delivery date/time

delivery_time                 Date/Time 8         task delivery time (time portion) - only relevant if greater than ordheader delivery date/time

update_date                   Date/Time 8         date of last manual update

scheduling                    Yes/No    1         temporary flag used during scheduling process only

Updated                       Yes/No    1         date/time of last manual update

Text                          Char      200       200 character comment for this task

UpdateSF                      Char      1         Y - pass changed details to SFDC system

MultiGroup                    Integer   2         for grouped tasks only holds the task number of the group task - see also job_type

MultiTskNum                   Integer   2         normally null - used to link duplex and triplex tasks that must be scheduled simultaneously

actual_endtime                Long      4         actual time task is completed-held as minutes from 1/1/1995

NumUp                         Long      4         number up

StepNum                       Long      4         step number used for link to costing system - normally the same as job

form                          Long      4         formno from pcwctask - appears redundant

MrQty                         Long      4         estimated quantity used in make ready process

CapIgnore                     Yes/No    1         if true then capability group scheduling is not used on this task

WastePerc                     Double    8         percentage of run waste

Report                        Yes/No    1         temporary flag used during report printing processes only

PrevTask                      Long      4         record number of previous task - set during assessment and used in scheduling

NextTask                      Long      4         record number of following task - set during assessment and used in scheduling

resav                         Char      1         summary of all resources for this task Y - available N - not available or pending - see resavdate

resavdate                     Date/Time 8         date last resource is expected to ba available

retainwaste                   Integer   2         0 = waste removed 1= run waste retain 2 = run & m/r waste retain - Radius I systems only

Held                          Yes/No    1         set to true if task is held- these tasks are not passed to sfdc

next_start                    Long      4         earliest start time of following task allowing for serial/parallel style, wait time and move time

resavstring                   Char      30        30 character string - one for each resource - as resav

ptaskstartdt                  Date/Time 8         planned task start date/time

ptaskenddt                    Date/Time 8         planned task end date/time

ataskstartdt                  Date/Time 8         actual task start date/time

ataskenddt                    Date/Time 8         actual task end date/time

ptaskstartyw                  Long      4         planned task start held in yyyyww format

ptaskendyw                    Long      4         planned task end held in yyyyww format

ataskstartyw                  Long      4         actual task start held in yyyyww format

ataskendyw                    Long      4         actual task end held in yyyyww format

origwc                        Char      12        original work centre set after work centre has been changed

followon                      Yes/No    1         set to true if task has been marked as followon and make ready time (mrlength) set to zero

matavail                      Char      1         summary of material status A = no material N = not actioned P = ordered R = to be called off T = called off Y = available Z = issued

p_lateststrt                  Long      4         calculated latest start date/time of task to achieve promised date - see also lateststrt

DoAssess                      Yes/No    1         short term flag mostly used by external processes (PsaConnect) used to indicate that this task must be reassessed when next opened in PSA

kstatus                       Char      1         SFDC - planned status  X or null  not started H - Held; S  Started; C - Complete (once status has been aligned to complete then kstatus is set to @)

 

runstat                       Char      1         SFDC - running status - X - Not Started; M - Make Ready; G - Good Copy; B - Bad Copy; S - Sort Copy; H - Held; C - Complete

startinit                     Long      4         SFDC - inital start time - held as seconds since 01/01/1990

starttime                     Long      4         SFDC - start time of current run - held as seconds since 01/01/1990

endtime                       Long      4         SFDC - end time of current run - held as seconds since 01/01/1990

elapsetime                    Long      4         SFDC - start time of current run - seconds

mkrqty                        Long      4         SFDC - recorded make ready quantity

runqty                        Long      4         SFDC - recorded run quantity

badqty                        Long      4         SFDC - recorded bad quantity

sortqty                       Long      4         SFDC - recorded sort quantity

outputqty                     Long      4         SFDC - recorded total throughput quantity

mrtime                        Long      4         SFDC - recorded make ready time - seconds

prodtime                      Long      4         SFDC - recorded good production time - seconds

badtime                       Long      4         SFDC - recorded bad production time - seconds

sorttime                      Long      4         SFDC - recorded sort time - seconds

stoptots                      Long      4         SFDC - total stops recorded

jobspeed                      Long      4         SFDC - recent recorded running speed - units/hour

targetspeed                   Long      4         SFDC - speed required to achieved planned end time - units/hour

calcend                       Long      4         SFDC - calculated end time based on current speed and quantity - ignores non working period - held as seconds since 01/01/1990

machid                        Char      12        SFDC - work centre name

firsttran                     Date/Time 8         SFDC - date/time of first transfer from SFDC

lasttran                      Date/Time 8         SFDC - date/time of last transfer from SFDC

changeflag                    Char      1         SFDC

aligned                       Yes/No    1         SFDC - if true then planned start and end have been aligned with latest SFDC actual running details

TreatAsPlanned                Yes/No    1         if task has been held in SFDC and replanned in PSA it will be treated as planned rather than held

tech1..nn                     Char      20        20, 34 or 48 technical attributes for this task - may be entered manually or imported from user defined Radius source

resav1..nn                    Char      1         30, 25 or 20 user defined resources - flag indicating available- Y or not available - N - this may be associated with a date resavdate

resavdate1..nn                Date/Time 8         30, 25 or 20 user defined resources - if not available this indicates expected date

resavreq1..nn                 Date/Time 8         30, 25 or 20 user defined resources - if available this indicates achieved date

tsxfer                        Yes/No    1         if true indicates running details to be passed to SFjobdet

tsdate                        Date/Time 8         if true indicates date that running details were passed to SFjobdet

sfdate                        Date/Time 8         date of most recent transfer to SFDC system

holdrsn                       Char      30        text giving reason that task is held

duplex                        Integer   2         number linking this task to other(s) on the same component on a different work centre that must run in true parallel
