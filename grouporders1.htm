Group orders

This gives a limited ability for similar – based on work centres used – orders or tasks to be grouped for scheduling purposes.

The essentials of the method are
bring the like orders together on the Planned tab of Schedule Control and highlight them.

Right click on the selection and select Group.

This will load the selected order into the Group Tasks function – they are always sorted by process type and work centre – it is important that the default process sequences are correct.

You can select a group of task and press ‘Group’ but the usual method is to allow PSA3 to ‘Auto Group’ but before you do set the group details – serial group – the default - is where the tasks for each order run consecutively on each work centre. Parallel group is for business forms where a wider web physically allows tasks with the same cut off length to be run side by side. For serial group also specify the percentage make ready reduction for the second and any subsequent make readies

After grouping the tasks each group is allocated a colour and the new make ready times can be seen – these are treated as follow on for scheduling purposes.

Closing the screen and returning to Schedule Control gives you just the Group order replacing the original 3 orders

At this time the scheduled order will always be shown in black – when right clicking for the context sensitive menu you get an additional option to drill down to the underlying tasks
