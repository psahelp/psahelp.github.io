Before you start > Upgrading from PSA2

SQL Server version.

PSA will work with all versions of SQL Server from 2005 to 2016 or later. Express editions will also work but are not recommended other than for single plant sites. It may work with older versions but this has not been tested.

SQL Server Express editions are available as a free download from Microsoft and can be used. It does not have all the features of a full version and some parts such as backups and integrity tests will have to be set up manually. Obviously this is not recommended for multiplant sites but is in use at many single plant sites.
