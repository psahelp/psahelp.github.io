Customer tab

The Customer Maintenance screen allows customer details to be maintained, customers are normally added automatically by the PsaConnect software during the import process, if an imported order’s customer does not exist.

Separate colours may be selected for each customer. Pressing the Colour button opens a pallet where 48 predefined and 16 million user defined colours may be selected. At run time a contrasting ink colour will be used to keep text visible against any background colour

A default colour should be defined in the same way so that customers not having a defined colour will use the default.

The Update Orders button will apply the selected colours and default to all orders in the database. Do not use unless you are implementing a customer based colour regime. This process is not easily reversed.

In PsaConnect select ‘Customer’ as the order colour on the Transfer Options tab.

The Customer Allocations grid allows time to be allocated to one or more customers on one or more work centres. See the separate appendix for details.
