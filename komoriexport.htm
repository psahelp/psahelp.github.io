Komori export

A data export facility has been provided, to supply scheduling information to a Komori work centre control system. Data is exported in a flat file format, (see Appendix A for details) each time the PSA live plan is updated.

 The system can be configured to enable the Komori data export, using the system configuration screen.

The export will become operational, if the ‘Enable Komori Export’ check box is checked, the export dump file can be selected using the text box provided, an adjacent browse button has been provided, and the default is the current working directory and ‘USERINFO.TXT’.

Attribute Mapping – the export format expects some named technical details – these can be indicated by linking to a selected technical attribute.
