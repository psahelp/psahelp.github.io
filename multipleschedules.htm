Multiple Schedules

It is possible to open more than one plan at a time. This can be achieved by the following steps.

·      Create separate working directories for each plan. E.G. 'C:\Program Files\Scheduling Assistant\TestPlan' and 'C:\Program Files\Scheduling Assistant\LivePlan'.

·      Create 2 copies of 'psa3.ini'. E.G. 'TestPlan.ini' and 'LivePlan.ini'

·      Edit these 2 ini files using Notepad or similar and change the line 'WorkingDirectory=' to point to the appropriate directory created at step 1.

·      Place a Windows shortcut to PSA.EXE on the desktop, right click and select 'Properties'. The Target should say something like "C:\Program Files\Scheduling Assistant\Psa.exe".

·      At the end of this string add a space and the full path to one of the ini files. It should now look something like this "C:\Program Files\Scheduling Assistant\Psa.exe" "C:\Program Files\Scheduling Assistant\TestPlan.ini"

·      Create a second shortcut and point to the other plan.

·      Then change the icon titles to ‘PSA Test’ and ‘PSA Live’ or similar
