Tentative Order Replacement
Tentative order may be set up by the planner to act as placeholders in the schedule. When the live order is transferred from Radius press the Tentative button and all scheduled tentative orders will be displayed. There are a number of filter options. Once a tentative order has been selected pressing the OK button starts the Difference Check routine. At the end of the process the live order will have the schedule times of the tentative order which will be deleted.
