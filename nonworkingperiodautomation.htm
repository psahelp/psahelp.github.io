Non-working period automation

Non working periods can now be defined in terms of a shift pattern (variable repeat period) and will be automatically be laid down a user definable distance into the future and will be cleared up a fixed number of days after the effective date has passed. It is therefore only necessary to handle exception to the standard pattern.

Note that the process involves defining work centre availability rather than non availability. The information entered is converted into standard non working periods for display on the planning board. A sample pattern is described here.

·      From the main menu select Configuration/Tables then the Availability tab

·      Select ‘New’ – bottom left and enter a code and description – say ‘Pattern1’ and ‘Two shift, five day’

·      Enter the Next Start Date (normally a Monday), Repeat period (normally 7 days but up to 28 days acceptable) and Days to Apply (This is variable but should be a week or two more than the most distant order scheduled – typically 60 to 90 days). Do not extend further than necessary – there will be some impact on database size and performance if too many periods are defined.

·      Select from the list on the right the work centres that this pattern will apply to. Once several patterns have been laid down the filter buttons at the bottom of this list can be used to help choosing affected work centres.

·      Now press the ‘New’ button above the time strip. This will create a new ‘availability period’ for day 1 (Monday in this case) starting at 8:00 hrs and ending at 16:00 hrs.

·      Using the left mouse button the edges of this block may be dragged to the required time or date. In this example drag the left side to 6:00 hrs, the right edge to 22:00 hrs and the bottom to day 4. The start and end time are shown during the drag process.

·      Repeat the last two steps this time dragging the entire block to day 5 and setting the start and end times to 7:00 hrs and 20:00 hrs. There are button above to help with fine adjustments of the start and end times

·      Press ‘Apply’ – this will save the definition and then run the housekeeping process to apply the changes and update these to the live schedule. The Next Start date will have advanced.
