Order Maintenance

There have been a number of minor revisions on this screen to improve information display. The Copy Order and Change Order Number utilities have moved to buttons at the bottom of the screen. All other options are remain available but from the Main Menu or Planning Board Menu.
Editing of technical attributes is much simplified.
The 3 estimating Analysis code from Radius I systems are available. User definable titles are set up under Configuration/System where the analysis set (A – Z) should also be defined. There is also a flag in PsaConnect that must be set to transfer this information.
Component tab – as discussed elsewhere the resource information has been considerably enhanced and now shows just the effective resources for each task of the component.
Materials – the layout has been changed to make it more readable. If linked to a Radius II system it will also show material size.
An order can be selected by clicking on the required entry in the order list and then clicking on one of the detail tabs for the required detail level.
On completing individual tasks, if all other tasks have been completed then the planning status of the order header will automatically set to completed.
The 'Next' and 'Previous' order buttons are available from any tab.
The screen may be resized by dragging one of the borders or corners – the new size and location are remembered.
