Supervisor functions

All operators marked as ‘Supervisor’ have additional facilities to enquire on the data that is currently being processed for all operators.

The Task Log button opens a new windows showing a record of past and current events that have been recorded on the system and have been/will be transferred to the costing system

It is possible for the supervisor to update non-key fields (quantity, status code and elapsed time) on these records.

The Op Log button opens a window showing a record of the current events since the last log on for a specific operator that was selected.

 

It is possible for the supervisor to update non-key fields (quantity, status code and elapsed time) on these records.
