Planning Board

A new shunt option has been added. With the ‘shunted drag-drop’ option turned on drag a task using the right mouse button. Provided the task remains on the same work centre then when dropped all tasks in the direction of drag will be shunted until the next gap in the schedule stops the process. This allows the schedule to be adjusted to the current time or a space to be created in which to drop a new task.

The shift (non working period) extend forward routine allows new periods to be added to existing periods. This is particularly useful when adding details for bank holidays. Any consequential duplication or overlap with existing periods is automatically detected and corrected.
