Before you start

First Time Running

All users have unrestricted access to the main menu screen. When opening a 'Plan' database he/she must supply a User name and entry password, a default user of ‘User1’ and password of  ‘123456’ has been provided which allows full access to all the system features at ‘Supervisor’ level. It is recommended that this user/password is replaced by one of your choice; the Supervisor level login allows access to password maintenance and other system configuration menus. (See Security).

On running the Scheduling Assistant for the first time the system will automatically be configured to have the recent list of ‘Plan’ databases set to the sample database supplied with the system (Sampl145.psh).
