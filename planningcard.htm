Planning Card

The Planning Card tab shows just 5 fields but these are possibly the most important for limited access users
Order Notes – the order header notes are repeated here and can be maintained – they apply to all tasks
Component Notes – component level notes apply to all tasks on the same component – generally either the printing or finishing route – and can be maintained here
Task Notes - apply only to this task and are also maintainable

Technical Attributes – those shown here relate to the task and therefore contain correct details of sheet size, inks, varnishes etc – if an attribute is blank a common header level attribute will be shown

Resources – all the resources applicable to this task whether they are apply by being global, apply at the component level or apply just to this task are displayed and maintainable here. Update can be controlled so that individual users can only update the attribute specific to their area.
