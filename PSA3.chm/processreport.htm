Process report

A new report has been provided to enable users to track the process of individual orders. Up to four process types can be selected, which the report will query to determine  their comprising tasks running status at each selected process. The format of the report is much like the ‘Work to List’ with the exception of resource information.

Tasks will be reported which appear within the specified date range, and which are designated to run on either a selected work centre or display code.

·      Process Report – generally used for finishing departments where existing progress can be shown by selecting suitable process types for the ‘completed through’ section
