Custom Reports

All reports on the scheduling assistant utilise ‘Crystal Reports’ and run (.rpt) files. A custom reports screen has been provided so that customers can run their own ‘Crystal Reports’ generated files from within the PSA.

Users should create their own ‘Custom’ directory within the scheduling assistants own ‘Reports’ directory into which all personal reports files should be located. The custom reports directory can be set and stored using the  button to select a custom report in the custom directory, all the Crystal report files will be automatically listed.

The custom report is run by selecting the required report from the list and pressing the ‘Print’ or ‘Preview’ buttons as required.
