Parameters tab - Plan and user offsets from company time

This area allows plants in different time zones to be co-ordinated. All times in each plan are local. By defining the plant offset from the main company time zone and the offset of a user from the same time zone it is possible for a user to open a plan and see the time line as a ‘local’ user would see it. Where jobs are linked between plants in different time zones this allows a task finishing in one plant at say 7:30 Central to be followed by a task starting at 6:00 Pacific.
