Group orders

This is work in progress – there are more detailed definitions elsewhere. It is not possible to add tasks to an existing group or change the order number and/or description

This function works at two levels – A) a system to enable multiple tasks on a work center to be combined and manipulated as a single task and B) a function to group similar orders (say multiple near identical orders for the same customer) and keep together all the relevant tasks through the plant (plate making, press, finishing).

The system is usable with some limitations but there are some desirable functions that are not yet available – these will be commented on later. If SFDC is in use I believe you will require the RRD version because this should recognise group tasks passed from PSA and treat them as grouped tasks created with SFDC. Only details of the underlying tasks ever get to costing so there should be no implications outside the PSA/SFDC area.

To turn the function on select ‘Allow task and order grouping’ from the Functionality tab of Configuration/System and update the change.
