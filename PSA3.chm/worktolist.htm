Reports > Standard Reports

Work to list

This option allows allows the standard WorK To List report to be printed by selecting the start and end dates rather than taking the planning board defaults. The result is otherwise similar. However an important extra feature is that an entire display code may be reported at the same time. If the ‘Group by Work Centre’ option is checked the standard report is used with a new page for each work centre. If unchecked then a slightly modified report is used that includes the work centre in the printed data.

The ‘Without Time Information’ option suppresses the planned start and end times.
