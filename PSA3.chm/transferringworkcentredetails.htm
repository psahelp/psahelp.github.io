Transferring Work Centre Details

 

Work Centre Details can also be transferred from the Radius database from then ‘pcworkc’ table to the Gui Scheduling ‘resource’ table.

 

 

Fields Transferred:

 

            Work Centre Short Name.              pcworkc.kwcsn          resource.resource

         Description                           pcworkc.description    resource.description

         Running Speed                             pcworkc.planning-speed      resource.defspeed

         Make Ready                         pcworkc.default-mr            resource.def-mr

 

 

 

Note: The Process sequence field in Gui is always defaulted to 1.
