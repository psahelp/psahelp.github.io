Actioning selected tasks

Using the mouse it is now possible to drag these selected tasks up or down until the tasks are in the desired sequence.

Alternatively with tasks selected the buttons at bottom of the screen may be used. These enable the selected tasks to be moved up or down as a group, to be firmed/unfirmed as a group or to be scheduled as a group starting from the scheduled time of the first selected task. During this process non working periods and any other firm tasks are respected.
For tasks to be firmed/unfirmed as a group they must all have the same status – a warning is displayed if they do not.
