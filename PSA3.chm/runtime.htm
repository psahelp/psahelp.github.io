Run Time

Once tooling details have been set up and some tool data has been imported there are 3 further automatic changes that will happen in PSA.

Order Maintenance gains an extra tab

The duplicates function gains an extra checkbox ‘Show Tools’ that when selected finds all the tool details.

The top section shows tool types and selecting one of these will show all the tool codes visible in the lower section. The lower list can be expanded to show individual tools where appropriate

Checking one of the lines in the lower area highlights the tasks using that tool
