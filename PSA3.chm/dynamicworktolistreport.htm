Dynamic Work To List Report

To enable this feature create a report named A4_dworklist.rpt or Lt_dworklist.rpt depending on your paper prefix in the Custom Reports directory. When this report exists a print button is enabled on the screen.

When you press the print button PSA will output each line of data from the Dynamic Work To List will be output to the TaskReport table.

The fields ordernumber, component and job are always populated to make easy linking to other PSA tables such as ‘job’ so that other fields can be included in your report. The data from 36 columns are output into fields res1…30 and resav1…6. In addition field resav30 contains the work centre name, resav29 contains the number of hours that has been declared as the firm zone, resav28 contains the number of hours inside the firm zone, resav27 contains the number of hours outside the firm zone.
