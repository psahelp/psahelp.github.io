Display Codes

A display code is a collection of work centres which can be selected on entry into the planning board, as a group of work centres for reports or as a filter for Schedule Control.

Any number of work centres can be placed into a display code.

For the Planning board work centres should normally be placed in process sequence order where possible so that a typical Gant chart view of the planning board is shown with order proceeding from top left to bottom right.

This is now simplified by the Up and Down buttons.

Any number of display codes may be set up – they are for the benefit of users of the PSA system.
