Configurable details

The sequence of columns is changed by opening the ‘Selected’ group and dragging an entry into the required position.
The widths of columns can be changed using the left mouse button between cells in the title row to drag the right edge of the column to a new width.
Clicking the right mouse button in the title row brings up a list of 3 options.
‘Lock/Unlock’ - enables the setting of up to 5 columns on the left as a non scrolling area.
‘Reset’ - restores the initial column widths and sequence. It will also revert all titles to their original values – this can be useful if new technical attributes have been added’
‘Title’ - enables the user to set their own column title – particularly useful if a column width has been reduced and a shorter title is required. All changes are remembered until ‘Reset’ is selected’
The other options are for resequencing rows by sorting and are only available after a number of rows have been selected.
