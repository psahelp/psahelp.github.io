Utilisation

The utilisation report now offers the ability to view or report on a week by week basis. As part of the configuration for this option all calendars can display the week number. The provides information relating to work centre usage throughout a factory. The number of production hours or machine loading is calculated along with the amount of free time available. A standard report together with a graphical representation is available.

If report by week number is checked the range dates are adjusted to reflect the start and end of actual weeks as defined under system configuration.

Include earlier orders will include tasks from previous weeks in the first week of the selected range.

If Adjust for efficiency is ticked the scheduled times are reduced by the efficiency factor of each work centre.

The ‘Data’ button can be used to reveal the actual figures used to produce the graph. A printout can be obtained using the ‘Print’ button.

Note:-  the report is still based on the original schedule start and end times which may be unreliable after manual task update. However new fields have been added to the task table holding scheduled start and end times as well as actual start and end times as captured from either manual task update or the planning board align option from SFDC. These dates are held in readable DMY,HMS format as well as Year/Week format to assist users to extract the data they require using Crystal Reports or similar.

Due to the complex nature of the manual and SFDC updates this data gathering process will require refinement. It should become possible to compare scheduled against actual enabling a more accurate efficiency factor to be calculated for each work centre.
