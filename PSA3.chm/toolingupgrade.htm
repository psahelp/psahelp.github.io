Tooling Upgrade

The old Radius system had fixed tool statuses. Recent versions have user define status. Because of the 8 status limit in PSA it was sometimes necessary to double up status values like this

This version will automatically remove the duplication by creating separate lines for each status – three actions may be required

·    Make sure that the sequence of statuses top to bottom reflect worst to best as PSA uses this sequence to find the worst case status when there is more than 1 tool

·    Update the description to remove duplicate values

·    Update the colour to remove duplicate values

If the latest version of Con8 is in use dropdowns will be available to select status values and descriptions imported from Radius
