Appendices including technical notes

PSA to SFDC - Export

 

Task information can be passed to the Shop Floor Data Collection Database. This becomes a task available to be run within SFDC.

 

Task:

The field sfplan.changeflag is set to 'N' or 'M' for 'new' and 'modify'

Task order number.               job.ordernumber                               sfplan.kjobcode

Task component number.       job.component                          sfplan.kcompno

Task task number.         job.job                                    sfplan.kprocno

Component description.  component.description                       sfplan.description

Order description.          ordheader.description                 sfplan.kjobsn

Order customer.            ordheader.customer                  sfplan.customer

Task description.            job.description                          sfplan.taskdesc

Task workcentre.           Job.resource                            sfplan.mach-id

Task runstatus.                     Job.runstatus (normally set to 'X')       sfplan.runstat

Task status.                  Job.status                                sfplan.kstatus

Task latest start time.            Job.lateststrt                                    sfplan.lateststart

Task planned start time. Job.strtime                               sfplan.kstarttime

Task planned end time.   Job.edtime                               sfplan.schedend

Task quantity.                       Job.quantity                                     sfplan.quantity

Task make ready quantity.      Job.mrqty                                sfplan.mrquantity

Task make ready time.   Job.mrlength                            sfplan.mrtime

Task form number.        Job.form                                  sfplan.formno

Task step number.         Job.stepnum                             sfplan.kstepno

Task number up.           Job.numup                               sfplan.noup

Task run speed.             Job.speed (or a calculated value) sfplan.schedspeed

Task waste percentage.  Job.wasteperc                          sfplan.wastepercent

The waste percentage transfer is conditional on Radius version
