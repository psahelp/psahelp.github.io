New plan

This utility can be used when setting up a new plan database. An existing database can be used as a template and selective data can be retained for use in the new database.

The new database will retain information according to the options selected. This can save time setting up new shift patterns and table information. This is useful for taking copies of the current plan or populating a temporary plan so that extensive ‘what-if’ scenarios can be tried
