Context sensitive options

Right clicking on a task in the ‘Scheduled’ column brings up a context sensitive menu that allows
Task Start Times - the calendar so that a new task start date/time may be entered. This defaults to the currently scheduled time
Order Maintenance – immediate access to the Order Maintenance screen for this order
Task Details – immediate access to the Task Maintenance screen for this task – warning – do not change the work centre as this will make the Dynamic Work List invalid
Materials – immediate access to the Order Maintenance screen on the Materials tab for this order
Move To – where the list of tasks is long – several screens – and you wish to move a task a long way up or down the ‘Move To’ option opens a separate list – selecting a task from this list will insert the task immediately before it.
Fix Point Schedule – allows other tasks of this order to be rescheduled based on the new location of the selected task
Set or Clear Follow On – allows the ‘follow on’ flag to be set or cleared – the make ready time will be reduced by the percentage specified for the work centre
