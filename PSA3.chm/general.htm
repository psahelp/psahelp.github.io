Screen by screen descriptions > Planning Board Preferences

General

Before entering the planning board the user must select their viewing preferences. These include which work centres they which to view, the start date and the ‘horizon’ or length of time represented by the width of the board.

Work centres can be selected by specific display code, which will select all the work centres in a given display code (see display codes in configuration). Alternatively the work centres for a specific order can be selected and only work centres relevant to the comprising orders tasks will be displayed. Finally the ‘Full Factory’ option will display all work centres present on the system.

Show Completed Tasks – this checkbox allows completed tasks to be displayed on the planning board. They will be coloured white and only limited options are available (e.g. they can’t be dragged anywhere)

Maximum horizontal scrolling – if the visible horizon is 7 days then 7 days are plotted on screen. Turning on this option will widen the plotted planning board by a factor of between 10 and 17 depending on screen resolution. The time taken to plot will obviously increase as many more tasks are visible but after that horizontal scrolling often does not require replotting at all.

Maximum vertical scrolling – depending on screen resolution only 13 to about 25 work centre are normally visible – less if more than line of text is displayed on each row. Turning on this option will heighten the plotted planning board until all work centres are plotted regardless of how many vertical lines are selected. Again, the time taken to plot will obviously increase as many more tasks are visible but after that vertical scrolling does not require replotting at all. There is a limit that can be reached requiring replotting but something like 100 work centres can be displayed with 7 or 8 lines of text without this limit being reached.

Feet/Metres and Copies – these radio buttons switch quantity display between Length and Impressions on web based tasks. There are still some teething problems in this area when copies are selected

‘Configure’ button can be pressed to allow further configuration options to be selected. New display codes can be created if required.
