Order Update - Export

 

Order information can be passed back to the Radius Database provided that the order originated here, Radius tables pcjob,pccomponent and pcwctask are updated. If a particular work centre does not yet exist on the Radius system then it is created from details held on the Gui system and a pcworkc record is created.

 

Order Header:

 

          Order Description.              ordheader.description         pcjob.description

        Order Quantity.          ordheader.quantity                    pcjob.quantity

        Order Plan Status.             ordheader.status                       pcjob.kstatus *

        Order Comments.              ordheader.txt                    pcjob.comments

        Order Delivery Date.   ordheader.deliverydate               pcjob.krequired

        Order Technical Atts.  ordheader.tech1/20            pcjob.gui-tech-att[20]

 

 

Component:

 

          Component Description              component.description        pccomponent.description

        Component Structure         component.complevel pccomponent.ksequence

        Component Status                     ordheader.status               pccomponent.kstatus

 

 

Task:

         

       Task Description                      job.description                          pcwctask.description

      Work Centre                   job.resource/ resource.resource          pcwctask.kwcsn

      Task Plan Status                      job.status                                pcwctask.kstatus

      Task Quantity                  job.quantity                                     pcwctask.quantity

      Quantity Completed          job.quantity_completed                      pcwctask.qty-cmpl

      Wait Time                               job.taskwaittime                       pcwctask.waittime

      Move Time                              job.taskmovetime                             pcwctask.move-time

      Run Time                        job.runlength                            pcwctask.run-time

      Make Ready                    job.mrlength                                   pcwctask.mr-time

      Latest Start                    job.lateststrt                                   pcwctask.latest-start

      Scheduled Start                      job.strtime                              pcwctask.kstart-time

      Scheduled End                job.edtime                                pcwctask.planned-finish

      Number Up                     job.no-up                                 pcwctask.numup

      Make Ready Qty                      job.mr-quantity                        pcwctask.mrqty

      Form Number                  job.formno                               pcwctask.form
