Order Information

An order based reports screen has been provided which is accessible from both the order maintenance screen menu and from the reports menu in the main screen. The order based reports screen provides a selection of order specific reports that can be previewed from a window. Thereafter reports can be printed or saved to a file.

The required report should be selected using the radio buttons, together with any preferences such as sorting order or status filtering, the report can then be run by pressing the ‘Report’ button.
