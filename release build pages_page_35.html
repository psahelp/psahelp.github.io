Reports
PSA currently has some 28 standard reports. These are supplied in A4 and Letter format and should be found in the directory pointed to by the ReportDir local parameter.
Do not change these reports and do not add to the reports in this directory as they are liable to be overwritten or lost in the event of a re-install of PSA. The ‘custom’ reports directory pointed to by the CustomReportDir local parameter is designed to handle all the options for custom reports.
All the reports are now distributed in Crystal 11 format and this is the minimum level that is likely to work correctly. PSA3 can handle reports written in Crystal 9, 11, 11.5 and .Net. No Crystal 9 runtime code is distributed with PSA but suitable run time files can be downloaded and installed.
To create your own version of a standard report see ‘Custom Reports’ below.
