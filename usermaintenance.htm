User Maintenance

The User Maintenance screen can only be accessed at Supervisor level, and is available through the Configuration – Users menu option.

The Scheduling Assistant has been provided with security measures with certain functions being password protected together with overall entry to the system. The level of access given to a user is controlled by their user name. There are four levels of system access, denoted by ‘Supervisor’, ‘Fully Amendable’, ‘Limited Access’ and ‘Enquiry Only’.

The ‘Supervisor’ password level gives the user access to all functions and menu options, including the System Configuration, and User Maintenance Screens.

The ‘Fully Amendable’ level provides a tailored level of access depending upon the users requirements. It does not allow access to user maintenance. This level is normally used for schedulers.

The ‘Limited Access’ level provides a lower level of access. It does not allow access to user maintenance, configuration, schedule control, etc. Access can be adjusted from almost nil to quite advanced depending upon the users requirements. This level is normally used for all users except schedulers who are required to update some part of the system.

The ‘Enquiry Only’ password level provides truly view only access with no updates possible at all.
