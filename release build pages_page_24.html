Unplanned tab actions
Any order in the ‘Unplanned’ grid can be planned by first selecting the order and pressing the ‘Plan’ button, the ‘Plan Order’ screen is then loaded. Normally if the whole order is to be planned i.e. the entire quantity then all the default entries can be accepted and the ‘OK’ button pressed. The description and notes text can be changed if required. Alternatively the order may be planned in part. If required, the new planned quantity should be entered in the ‘quantity’ text box; the system will calculate any remaining quantity and adjust the original order accordingly when the ‘Apply’ button is pressed. A new order description and delivery date can be entered if required.
Any orders planned in this way will be treated as separate from the original order, but will be given a new order number derived in part from the original number and given a suffix denoting the number of ‘quantity splits’. For example if the order is planned in part for 20000 its new order number will be ‘99/1’, the original order number will remain as ‘99’ and its quantity change to 80000, the remaining quantity. Once the required quantity has been planned, the order will move to the right hand grid where it can be re-sequenced if required.
