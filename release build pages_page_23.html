What If and AutoSync modes
Until now PSA has worked in the same way for all users. This method remains completely unchanged for What-If users. It is therefore possible for all users to continue working in exactly the same way as before.
However this method had several clear disadvantages – a user could be looking at a planning board that is several hours out of date and equally - changes made by that user are not available to anyone else until saved – again a possible delay of several hours. Combining these delays just makes possible errors worse and many of the advantages of a screen based planning system are lost.
