System Configuration

The System Configuration Screen is accessed via the ‘Configuration – System’ Menu options in the main screen, a ‘Fully Amendable’ level password must be supplied before access is permitted. This screen is used to configure the operational preferences of the system on the current PC; some of the data is retained in the Psa3.ini file located in the Windows directory
