SFDC to PSA - Import

 

Running task information can be passed back to the PSA Database and is held in a separate table exclusively for running details.

 

Task:

Task order number.                       sfjobdet.kjobcode                     job.ordernumber

Task component number.               sfjobdet.kcompno                     job.component

Task number.                        sfjobdet.kprocno               job.job

Task stop reason code.           sfjobdet.statcode                      job.statcode

Task status.                          sfjobdet.kstatus                job.kstatus

Task run status - good, bad etc.      sfjobdet.runstat                job.runstat

Task actual start time.                    sfjobdet.starttime                     job.starttime

Task planned end time.          sfjobdet.endtime                       job.endtime

Task calculated end time.               sfjobdet.calcend                        job.calcend

Task running time.                 sfjobdet.elapsetime                   job.elapsetime

Task make ready count.         sfjobdet.mrqty                         job.mrqty

Task good copy count.                     sfjobdet.runqty                         job.runqty

Task bad copy count.                       sfjobdet.badqty                        job.badqty

Task sort quantity.                   sfjobdet.sortqty                        job.sortqty

Task input quantity.(not used)           sfjobdet.inputqty                               job.inputqty

Task output quantity.(total)               sfjobdet.outputqty                             job.outputqty

Task make ready adjustment.           sfjobdet.mradj                         job.mradj

Task run adjustment.                       sfjobdet.runadj                         job.runadj

Task bad adjustment.                      sfjobdet.badadj                        job.badadj

Task sort adjustment.                      sfjobdet.sortadj                        job.sortadj

Task make ready time.           sfjobdet.mrtime                        job.mrtime

Task time on good copy.         sfjobdet.prodtime                              job.prodtime

Task time on bad copy.          sfjobdet.badtime                               job.badtime

Task time in sort.                  sfjobdet.sorttime                              job.sorttime

Task total stops.                    sfjobdet.stoptots                       job.stoptots

Task current speed.                       sfjobdet.currspeed                            job.currspeed

Task planned speed.                      sfjobdet.jobspeed                             job.jobspeed

Task status date changed.               sfjobdet.statdate                               job.statdate

Task target speed.                 sfjobdet.targetspeed                  job.targetspeed

Task work centre.(1.35 only)           sfjobdet.mach-id                               job.machid
