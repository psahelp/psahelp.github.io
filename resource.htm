capability                    Char      6         capability group - optional

proSequence                   Integer   2         process sequence

mr_time                       Integer   2         redundant

defspeed                      Double    8         default speed - used for new tasks and task moves between work centres

def_mr                        Long      4         default make ready time

def_suffix                    Char      5         redundant

Short_Des                     Char      20        short description

DefMVtime                     Long      4         default move time

Upstatus                      Long      4         update reference number

KMS                           Integer   2         used when linking output to Komori work centres

EFactor                       Integer   2         efficiency factor - default 100

WarnOnOvers                   Yes/No    1         if true completing task with completed quantity exceeding required quantity by overslimit will trigger warning

OversLimit                    Integer   2         held as percentage

usepacking                    Yes/No    1         when linked to Radius I systems packing materials will be linked to tasks on this work centre rather than the first task of the component

UseInfinite                   Yes/No    1         all scheduling on this work centre assumes infinite capacity - use for outwork processes

defParallel                   Yes/No    1         if true new tasks will be set to parallel style

defwaittime                   Long      4         default wait time

orderdeleteignoretasks        Yes/No    1         not in current use

CustomWorkList                Char      20        name of custom Work To List report to be used for this work centre

analcode2                     Char      30        reserved for future use

v_ptype                       Char      10        Radius process type (A100, P200 etc) - not always loaded

to_sfdc                       Yes/No    1         reserved for future use

freezeactive                  Yes/No    1         turns on dynamic firm line for tasks on this work centre

freezemoving                  Yes/No    1         if true the dynamic firm line will stay x minutes ahead of 'now'

freezetime                    Long      4         records firm line position

Generic                       Yes/No    1         if true this work centre is generic and an attempt will be made to transfer tasks to a non generic work centre in the same capability group

CostRate                      Double    8         used in determining best work centre during transfers from generic

digits                        Integer   2

counter                       Long      4
