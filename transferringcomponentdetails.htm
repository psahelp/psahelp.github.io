Transferring Component Details

 

Transferring data from the Radius Component table ‘Pccomponent’ to the Gui Scheduling Component  table ‘Component’.

 

 

        Company No.            kco                  Component.company_no

        Order Number.          kjobcode           Component.orderno

        Component No.         kcompno           Component.compnumber

        Description.                      description        Component.description

        Structure Level.         ksequence         Component.complevel

 

 

 

The Structure Level Field is used to determine the order in which components are scheduled. In Gui scheduling a  Component Relationship table is built, based upon an orders respective components and their structure levels, creating details of Parent and Child components.
