FIELD NAME                    TYPE      LENGTH    DESCRIPTION

 

ordernumber                   Char      15        Radius production order - trailing @ indicates duplicate order reimported with changed details

Description                   Char      40        Description

status                        Char      1         U - Unplanned, P - Planned, F - Firm, C - Complete

customer                      Char      25        Customer name detail only

deliverydate                  Date/Time 8         Date portion only - also referred to 'required have ready date'

tech1..nn                     Char      20        20, 34 or 48 technical attributes entered in estimating - now used in Komori link, capability group definition

scheddate                     Date/Time 8         Date portion of completion of last task

promdate                      Date/Time 8         Date portion of promised date

qty                           Long      4         Order quantity

colour                        Char      10        Display colour of task bars (1 magenta, 2 , 3 light blue, 4 , 5 red, 6 , 7 , 8)

sequence                      Long      4         Set in sequence Manager. When autoscheduling multiple orders determines priority

txt                           Char      255       Comments - now 255 characters

type                          Char      1         Live, Tentative, Undetermined

del_time                      Integer   2         Time portion of delivery date - usually zero - held as minutes since midnight

sched_time                    Integer   2         Time portion of completion of last task

prom_time                     Integer   2         Time portion of promised date

assessed                      Char      1         Y if assessment schedule has been run

tentative                     Char      1         Y if order is Tentative - do not pass to SFDC

job_type                      Char      10        used to indicate order number of group order to which this order belongs

updated                       Char      1         set to 1 indicates order has changes to be passed back to Radius

pecas_orderno                 Char      15        redundant

DefOrd                        Yes/No    1         Only one order can be default - used as template when importing orders without tasks from Radius

TentativeDate                 Date/Time 8         Date that tentative order was last scheduled

TentativeTime                 Date/Time 8         Time that tentative order was last scheduled

Upstatus                      Long      4         update reference number

Report                        Yes/No    1         temporary flag used to indicate to Crystal report this order must be selected for reporting

estimate                      Char      10        Estimate number from which order was created

mestimate                     Char      10        Master estimate number - Radius 1.35 and up

CHRDreason                    Char      200       Confirm Have Ready Date change reason

CHRDRdate                     Date/Time 8         Date that Confirm Have Ready Date changed

security                      Integer   2         0 - None, 1 - Secret, 2 - Confidential

PressApp                      Integer   2         0 - None, 1 - Colour, 2 - Copy, 3 - Colour and Copy

OrderCat                      Integer   2         0 - None, 1 - New, 2 - Rework, 3 - Repeat

origkorder                    Char      30        original sales order from Radius I system - customer reference from Radius II

InvCopy                       Yes/No    1         Invoice Working Copy - transferred from Radius I only - can be set manually in PSA

ProForma                      Yes/No    1         Pro Forma Invoice - transferred from Radius I only - can be set manually in PSA

ConsOrder                     Yes/No    1         Consignment Order - transferred from Radius I only - can be set manually in PSA

IndPslip                      Yes/No    1         Individual Packing Slip - transferred from Radius I only - can be set manually in PSA

FromPecas                     Date/Time 8         Date of most recent order transfer from Radius system

PecasDeleted                  Yes/No    1         set by PsaConnect to indicate that order appears to be deleted in Radius system - filter available in order maintenance - check Radius status

blackink                      Yes/No    1         used to set ink colour to black or white to contrast order colour - obsolete - ink now determined automatically

repjobcode                    Char      15        transfer from Radius II - holds the jobcode that this order replaces. Treated as for @ suffix orders

origDelDate                   Date/Time 8         original delivery date if changed in PSA

CompletedDate                 Date/Time 8         date that order achieves completed status - used in housekeeping to retain completed orders n days

orderdate                     Date/Time 8         date order created - Radius I only

ifvcr                         Date/Time 8         customer specific - date initial firm version created

ifvby                         Char      20        customer specific - user who created initial firm version

ResRules                      Char      240       customer specific - list of rule codes used in initial firm version

ordercatdesc                  Char      30        Radius I customer specific plate-type Radius II pv_sorderitemtype.soitemtype

analcode1                     Char      30        Radius I only - analysis code - some Radius II - account executive

analcode2                     Char      30        Radius I only - analysis code

analcode3                     Char      30        Radius I only - analysis code

firstcalloff                  Date/Time 8         Radius I only - first unfullfilled call off date for single item

ItemOnHand                    Double    8         customer specific - Radius I item quantity

Linked                        Char      1         Radius II only 1 - this order is linked to a prepress order 2 - this order is linked to a production order

LinkStart                     Long      4         scheduled start of linked order

LinkEnd                       Long      4         scheduled end of linked order

LinkPlant                     Char      8         plantcode of linked order

LinkOrder                     Char      15        jobcode of linked order

LinkStatus                    Char      1         status of linked order
