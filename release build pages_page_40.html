Custom Reports
If you wish to change any standard report simple copy it into the ‘custom’ directory, modify it as required and distribute to any users who require it. At run time PSA will always look for a ‘custom’ version of every report and run it before defaulting back to the standard report.
The most commonly modified report is the WorkList. As this typically contains resource and technical details that are specific to a process type (ink details on a press task, glue style on a gluing step) you can create modified versions for each process type or work centre. Having created the new report in the custom directory go to Configuration/System/Work Centre and enter the name in the ‘Custom Work List’ field.
