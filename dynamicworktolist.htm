Dynamic Work To List

This is activated by clicking one of the work centres on the planning board. You should first check that under the Menu option ‘Screen Mode’ ‘Show Dynamic Work to List’ has been checked.
This enables one or more scheduled tasks to be resequenced using simple ‘Up’ and ‘Down’ buttons. Tasks will be listed provided they appear in the current display horizon on the planning board if the ‘Show Limited Dynamic List’ option has been selected.
Column widths may be adjusted by dragging the column separator within the heading area.
The columns displayed and their sequence are selected from the Planning Board Entry screen under Configuration.
The available columns have increased massively to include Resources, Materials, Items, Notes and specials such as ‘previous task completed quantity, make ready reduction. The full list contains about 170 items and is grouped as shown above. Once the required columns are selected they can be dragged into the required sequence in the selected area above
Note also that separate selections can be made for each process type so that only items appropriate to that process need be selected. The column widths you define are also remembered at process type level
Up to five columns may be locked by right clicking within the heading area and selecting 'lock'.
This screen shows all tasks for the selected work centre including tasks that have not yet been scheduled – these will always appear at the bottom of the list.
Completed tasks are not shown.
If a task is running it will always appear as the first task on the list and will normally be locked there.
A row number indicates the original sequence before changes are updated by pressing OK.
