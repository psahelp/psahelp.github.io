password

FIELD NAME                    TYPE      LENGTH    DESCRIPTION

 

UserName                      Char      20        user name - free format

UPassword                     Char      6         password - encrypted

LoginTime                     Date/Time 8         date and time of last login

Enabled                       Yes/No    1         Yes - allow log on No - deny log on

TotalLogins                   Long      4         count of logins for this user

UMessage                      Char      200       text to display to user

UAccess                       Integer   2         base access level - 1 - supervisor 2 - full access 3 - read only + allow options

Upstatus                      Long      4         update reference number

AllowOrderMnt                 Yes/No    1         allow changes in order maintenance

AllowShiftMnt                 Yes/No    1         allow changes to non working periods

AllowCustMnt                  Yes/No    1         allow additions to customer file

AllowResUpd                   Yes/No    1         allow resource updates - see also resflags

AllowTaskUpd                  Yes/No    1         allow changes in task maintenance

AllowSentPoll                 Yes/No    1         allow polling of PsaConnect

AllowUtils                    Yes/No    1         allow access to Utility menu

AllowReports                  Yes/No    1         allow access to Reports menu

AllowTableMnt                 Yes/No    1         allow access to Tables menu

AllowSfeb                     Yes/No    1         allow sfdc running bars

AllowImportExport             Yes/No    1         allow hold and release of tasks and/or orders

UpdatePassword                Yes/No    1         if true password required on all updates

AllowUpdateMessaging          Yes/No    1         allow user to add notes to update process

AllowWhatIf                   Yes/No    1         allow whatif mode

AllowAutoSync                 Yes/No    1         allow autosync mode

DefaultAutoSync               Yes/No    1         default to autosync mode at log on

resflags                      Char      30        30 flags (Y/N) controlling user update access to individual resources

opcode                        Char      6         operator code - required for time sheet entry

allownotesmnt                 Yes/No    1         allow update to separate notes section

 

INDEX                         PRIMARY   UNIQUE    FIELDS

 

UPassword                     Y         Y         +UPassword

Upstatus                      N         N         +Upstatus
