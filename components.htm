Components

An order's components can be maintained in the ‘Components’ section of the order maintenance screen. Components require only description and quantity information; resources are maintained in the grid to the right of this screen. New component can be added by pressing the ‘New’ button. A ‘Note’ field has been added to the component and appears on the Resource report. This can be used to pass more detailed requirements to resource suppliers.
When maintaining resources a zero in the Task column indicates a ‘Global’ resource applying to all tasks. A ‘C’ in the column indicates a component level resource applying to all the tasks of the component. A number indicates the task to which this resource applies. If entry is from the planning board and a specific task has been selected then the task column will highlight the resources applicable to this task.

As each component is selected the resource information is displayed to the right. Click in the 'available' column to toggle the Y/N flag. Click in the date column allows direct date entry but for users who prefer to select dates by calendar a button appears to the right of the column to pop up a calendar.

On completing the changes the Apply or OK buttons should be pressed to save the changes and re-assess the component.
