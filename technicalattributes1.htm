Technical Attributes

The Attributes section of the form allows the planner to select which technical attributes will appear against each order on the system. Up to 20 technical attributes are allowed, they appear as labels against each order into which descriptive data can be entered.

Attributes can be added or changed by clicking in the grid; the attribute description can then be edited. Double Clicking on an attribute in the grid will allow the entry to be cleared. Press the OK button to save any changes you make.

If Radius Estimating is in use the technical attributes for an order can be entered there and the names used here should be the same or similar. These details will be transferred at order header level

The other columns allow up to 2 possible sources for task level technical attributes to be declared. Almost any information can be specified as a technical attribute and will be copied from Radius I or Radius II source data. The primary source files are the works instruction and item definition tables. Likely candidates are sheet or web and item sizes and weights, inks, gluing styles etc.

Fields may be selected from some 60 Radius II tables and about 20 Radius I tables. Earlier system have a much reduced list of available sources Because of the variability of how data is held in different versions of Radius selecting fields is very much a trial and error process unless a detail knowledge of where your form of estimating, work instruction and item details are held.

The list of available fields is constantly being extended so if you have recently updated PsaConnect then run a Special Transfer to Populate Attribute Field List and see what new tables have been added to the list.

To insert a new source first click the Default Source column in the required row – it should turn yellow. Then select the table and field to be inserted. In certain cases (particularly item coatings) you must also enter Part, Coat Type and/or number. Once sufficient details are defined the small OK button becomes available and should be pressed. Once all the attribute have been defined press the Apply button and update the changes. It may be necessary to stop and restart PsaConnect for the latest sources to become fully effective.

By right clicking on a source you can bring up a menu allowing you to ‘interpret’ values. These are normally of the form Value/Display where value might be a status – 0, 1, 2 etc and Display is that status in words – Not started, Running, Complete.
There is a special case shown below where entering the Value <format> allows you to enter any valid Microsoft format string for number (as per Excel) and if the input is a number it will be formatted as requested. Typical use is to strip off trailing zeroes where sizes are in millimetres
