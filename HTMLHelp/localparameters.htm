Local Parameters

A number of parameters are held in a file named PSA3.INI. This is stored in the C:\Windows directory. On an NT4 PC this is C:\WinNT

[Defaults]

WorkingDirectory=               File Path to Psa.EXE
LivePlan=                    File Path to Live Plan (usually a network location)
HelpFile=                     File Path to Help File (not currently available)
ReportDir=                  File Path to Standard Reports
CustomReportDir=        File Path to Custom Crystal Reports
TranslationDB=             File Path to PsaTrans.mdb (translation database)
DateSelect=Y
PrefCountry=United Kingdom
LastDeliveryTime=
