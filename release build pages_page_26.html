Utilisation
You can select a work centre or group of work centres here, plus a number of weeks (max 16). On the planned order tab the table at the bottom will show, for each work centre week by week, the number of available hours and the number of utilised hours. The utilised hours are adjusted for work centre efficiency and thus represent a summary of the planning board. However the figures also include planned but not scheduled orders on the same work centres. These are assessed on the basis of latest task start times to meet the delivery date (or promised date if entered). The ‘task time’, ‘from week’ and ‘to week’ columns show the efficiency adjusted time for that order on the monitored work centres and the weeks where that time is scheduled or expected to be scheduled.
The option to ‘Assess all task time in first week’ does what it says – all the task time is added to the from week column. Note that if there are very long running tasks this may well show the ‘from week’ as overloaded and the ‘to week’ as under loaded.
